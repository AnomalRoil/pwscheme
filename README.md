# pwscheme
Golang package defining password schemes

Supported schemes
- {SSHA}        Salted SHA1
- {SSHA256}     Salted SHA256
- {SSHA512}     Salted SHA512


Docs: https://godoc.org/github.com/jsimonetti/pwscheme
